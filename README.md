# ALMA Spa SaaS

Primera version navegable del CRM omnicanal para ALMA Spa.

## Que incluye

- Panel con metricas, pipeline y oportunidades calientes.
- Bandeja omnicanal para WhatsApp, Instagram y TikTok.
- Ficha de cliente conectada a cada conversacion.
- Agenda visual con citas.
- Catalogo de servicios, precios y duracion.
- Campanas y automatizaciones de seguimiento.
- Configuracion inicial del bot IA.
- Simulador de respuesta automatica.

## Como abrir

Abre el archivo `index.html` en el navegador.

No requiere instalacion, servidor ni internet.

## Siguiente fase recomendada

1. Convertir esta demo en app real con base de datos.
2. Agregar autenticacion y roles.
3. Conectar WhatsApp Business Platform.
4. Crear API para agenda, clientes, conversaciones y servicios.
5. Integrar IA con base de conocimiento editable.
6. Preparar integraciones de Instagram, Facebook y TikTok.
