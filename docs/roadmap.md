# Roadmap

## Fase 1: Demo validable

- Interfaz navegable.
- Datos simulados.
- Flujo de bandeja, agenda y clientes.
- Simulador de respuesta del bot.

## Fase 2: MVP real

- Login.
- Base de datos.
- CRUD de clientes, servicios y citas.
- Bandeja de WhatsApp con API oficial.
- Recordatorios manuales o semiautomaticos.

## Fase 3: Automatizacion

- Bot conectado a base de conocimiento.
- Clasificacion automatica de clientes.
- Seguimientos programados.
- Transferencia a humano.
- Reportes de conversion.

## Fase 4: Omnicanal

- Instagram y Facebook.
- TikTok leads o mensajes si la API disponible lo permite.
- Campanas por fuente.
- App movil o PWA.

## Fase 5: SaaS multiempresa

- Multi-tenant.
- Planes y pagos.
- Panel de superadmin.
- Plantillas por industria: spa, clinica estetica, salon, wellness.
